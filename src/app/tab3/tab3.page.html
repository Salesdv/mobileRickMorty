Este aplicativo foi desenvolvido como projeto de avaliação para a disciplina de Desenvolvimento para Dispositivos Móveis, utilizando o framework Ionic com Angular.

O app consome dados da API pública "The Rick and Morty API" para exibir informações sobre personagens da série animada Rick and Morty.
